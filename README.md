# nixos

home for my nixos system configurations

## Work

1. Apply `work.configuration.nix`
1. *As nixos user* run
   `systemctl --user enable auto-fix-vscode-server.service`
   `systemctl --user start auto-fix-vscode-server.service`
1. Voila
